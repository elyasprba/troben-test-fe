## 📌 How To Install

- Clone this repo

```
  $ git clone https://github.com/elyasprba/troben-test-fe.git
```

- Go to folder repo

```
  $ cd troben-test-fe
```

- Install module

```
  $ npm install
```

- To start the project

```
  $ npm run start
```
